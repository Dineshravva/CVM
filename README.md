# CVM - Stack-Based Virtual Machine, Compiler & Bytecode Disassembler

This project implements a custom stack-based virtual machine and compiler in standard C++20 using recursive descent parsing and lexing principles.

The project compiles a custom scripting language into bytecode and executes it using a stack-based virtual machine. It also includes an enhanced bytecode disassembler for debugging and understanding generated instructions.

## Features

## Project Structure

### Lexer
Converts source code into tokens.

Example:

```text
let x=10
```

becomes:

```text
LET IDENTIFIER EQUAL NUMBER
```

---

### Parser

Consumes tokens and generates an Abstract Syntax Tree (AST) using recursive descent parsing.

---

### Compiler

Traverses the AST and generates bytecode instructions.

Example:

```text
let z=x+y
```

can become:

```text
LOAD_VAR
LOAD_VAR
ADD
STORE_VAR
```

---

### Enhanced Disassembler

Converts generated bytecode into a human-readable representation.

Features:

- Displays instruction addresses
- Shows variable names with indices
- Displays stack effects
- Shows jump offsets
- Resolves jump targets
- Improved formatting for readability

Example output:

```text
===== DISASSEMBLY =====

0000   PUSH_INT          10   [push1]
0009   STORE_VAR         x(0)   [keep stack]

0024   LOAD_VAR          x(0)   [push1]
0026   LOAD_VAR          y(1)   [push1]

0038   ADD               [pop2->push1]

0069   JUMP_IF_FALSE     offset(+24) -> 98

=======================
```

Stack effect meanings:

```text
[push1]
→ pushes one value onto stack

[pop1]
→ removes one value from stack

[pop2->push1]
→ removes two values and pushes one result

[keep stack]
→ stack size unchanged
```

---

### Virtual Machine (VM)

Executes generated bytecode using a stack-based evaluation model.

## How to Build

Requirements:

- CMake
- C++20 compatible compiler

Examples:

- MSVC
- GCC 13+
- Clang

Create build directory:

```bash
mkdir build
cd build
```

Configure project:

```bash
cmake ..
```

Build:

```bash
cmake --build .
```

---

## How to Run

Windows:

```powershell
.\Debug\cvm.exe ..\script.cvm
```

or

```powershell
.\cvm.exe ..\script.cvm
```

---

## Example Script

```text
let x=10
let y=20

let z=x+y*2

print z

let count=0

while(count<5){

    print count

    count=count+1
}

if(z==50){

    print 999
}
else{

    print 0
}
```

Output:

```text
50
0
1
2
3
4
999
```
